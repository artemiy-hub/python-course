### –°—Ä–µ–∑—ã (Slicing)

–°—Ä–µ–∑—ã –ø–æ–∑–≤–æ–ª—è—é—Ç –ø–æ–ª—É—á–∞—Ç—å –ø–æ–¥—Å–ø–∏—Å–∫–∏:

### –í–Ω—É—Ç—Ä–µ–Ω–Ω—è—è —Å—Ç—Ä—É–∫—Ç—É—Ä–∞ —Å–ª–æ–≤–∞—Ä–µ–π

–ù–∞—á–∏–Ω–∞—è —Å Python 3.7, —Å–ª–æ–≤–∞—Ä–∏ –∏—Å–ø–æ–ª—å–∑—É—é—Ç **compact dict** —Ä–µ–∞–ª–∏–∑–∞—Ü–∏—é:
- –û—Å–Ω–æ–≤–∞–Ω–∞ –Ω–∞ —Ö–µ—à-—Ç–∞–±–ª–∏—Ü–∞—Ö
- –°–æ—Ö—Ä–∞–Ω—è–µ—Ç –ø–æ—Ä—è–¥–æ–∫ –≤—Å—Ç–∞–≤–∫–∏ —ç–ª–µ–º–µ–Ω—Ç–æ–≤

| –û–ø–µ—Ä–∞—Ü–∏—è | –°–ª–æ–∂–Ω–æ—Å—Ç—å | –û–±—ä—è—Å–Ω–µ–Ω–∏–µ |
|----------|-----------|------------|
| `dict[key]` | O(1) —Å—Ä–µ–¥–Ω–∏–π —Å–ª—É—á–∞–π | –•–µ—à–∏—Ä–æ–≤–∞–Ω–∏–µ –∫–ª—é—á–∞ |

### –ú–µ—Ç–æ–¥—ã —Å–ª–æ–≤–∞—Ä–µ–π

```python

# –ú–æ–¥–∏—Ñ–∏–∫–∞—Ü–∏—è
```

### –°–ª–æ–≤–∞—Ä–Ω—ã–µ –≤–∫–ª—é—á–µ–Ω–∏—è (Dict Comprehensions)

```python
# –ë–∞–∑–æ–≤—ã–π —Å–∏–Ω—Ç–∞–∫—Å–∏—Å: {key_expr: value_expr for item in iterable}
```

### –ü—Ä–æ–¥–≤–∏–Ω—É—Ç—ã–µ –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç–∏ —Å–ª–æ–≤–∞—Ä–µ–π

```python
for num in numbers[:]:  # –°–æ–∑–¥–∞–µ–º –∫–æ–ø–∏—é –¥–ª—è –∏—Ç–µ—Ä–∞—Ü–∏–∏
    if num % 2 == 0:
        numbers.remove(num)
```
